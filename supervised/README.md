# Supervised Learning

This section contains supervised learning projects where models are trained on labeled data to learn mappings from inputs to outputs. The projects are further organized into **regression** tasks (continuous outputs) and **classification** tasks (discrete outputs).

## Overview

Supervised learning forms the foundation of predictive analytics. These projects demonstrate:
- How to formulate prediction problems with labeled datasets
- Model training, evaluation, and selection strategies
- Regularization techniques to prevent overfitting
- Cross-validation and performance metrics appropriate for each task

## Subfolders

- **regression/**: Predicting continuous numerical values
- **classification/**: Predicting discrete class labels

---

## Key Concepts

### Regression
Predicting continuous outputs (e.g., life expectancy, house prices) using techniques like Linear Regression, Ridge, and Lasso regularization.

### Classification
Predicting discrete class membership (e.g., disease presence, survival outcome, image labels) using Logistic Regression, Decision Trees, Random Forests, and Deep Learning approaches.
